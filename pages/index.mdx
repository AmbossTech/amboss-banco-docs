# BancoLibre

BancoLibre is a self-custodial cryptocurrency web wallet that prioritizes your security while ensuring a seamless and user-friendly experience. It is designed to make creating and managing your wallet as straightforward as possible, right within our website, without compromising on security.

## Features

- **Multi-Currency Support**: Supports Bitcoin and Tether.
- **Lightning Network Integration**: Send and receive money via the Lightning Network.
- **Lightning Address**: Each account includes a Lightning address for easy payments.
- **Social Features**: Fully encrypted chat with your contacts.
- **Web-Based Platform**: Accessible from any device with internet access.

### What cryptocurrencies does BancoLibre support?

BancoLibre currently supports Bitcoin and Tether.

### Can I use BancoLibre for everyday transactions?

Absolutely! BancoLibre is designed to facilitate both everyday transactions and more advanced cryptocurrency activities, making it versatile and user-friendly.

### Is there a limit to how much cryptocurrency I can store on BancoLibre?

BancoLibre does not impose specific limits on the amount of cryptocurrency you can store, but entrusts you to adhere to best practices for security and storage.

### How do I create a new wallet with BancoLibre?

During the account creation process, BancoLibre creates an initial wallet. Later, you can create more wallets using the BancoLibre client.

### How does BancoLibre interact with the Lightning Network?

BancoLibre uses swaps provided by [Boltz](https://boltz.exchange/) to send and receive over the Lightning Network.

### Can I change my MIBAN Code?

It is not currently possible to change your MIBAN Code.

### How does the MIBAN Code work?

The MIBAN Code is using swaps behind the scenes to allow you to receive and send Lightning payments without needing to run a Lightning node.

### What happens if I lose my mnemonic or password?

Your mnemonic seed phrase is encrypted and stored on BancoLibre servers, which can be decrypted using your account password. If you forget your password, no one will be able to decrypt your mnemonic to recover any funds held.

### What is the role of the symmetric key in BancoLibre?

The symmetric key is used for encrypting and decrypting your sensitive data, ensuring that only you can access your private information.
