# Self hosting the MiBanco Frontend

> GitHub Repository [here](https://github.com/AmbossTech/amboss-banco)

## Running locally for development

MiBanco consists of 2 services: frontend and backend. This page will explain how to run the frontend by yourself.

### Prerequisists

- NodeJS (`v20.11.1`)
- npm

### Getting started

1. `git clone https://github.com/AmbossTech/amboss-banco.git`
2. `cd amboss-banco`
3. `npm install` to install needed dependencies
4. `cp .env.example .env.local`
5. `yarn dev` or `npm run dev`
6. app runs on `localhost:3000`

## Build for production

This is a [Next.js](https://nextjs.org/docs/app/building-your-application/deploying) application which can be easily deployed using a hosting provider like [Vercel](https://vercel.com/new?filter=next.js) or on your own server.
